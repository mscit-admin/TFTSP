# الحزمة الرقابية والامتثالية — منصة TFTSP

**منصة شجرة العائلات والقبائل (Tribal & Family Tree SaaS Platform)**
نظام SaaS متعدد المستأجرين لإدارة بيانات الأنساب الشخصية والحسّاسة للقبائل العربية، بعزل صارم بين المستأجرين.

---

## الغرض من هذه الحزمة

توثّق هذه الحزمة **الضوابط الأمنية والحوكمية الفعلية المطبَّقة في النظام** كما هي في الكود والتصميم، لا كما هي مأمولة. كل ضابط مذكور هنا مرتبط بموضعه الفعلي في قاعدة الكود (ملف/آلية/جدول قاعدة بيانات) حتى يتمكن المدقّق من التحقق منه مباشرةً. وحيثما لم يُبنَ ضابط بعد (مثل المصادقة متعددة العوامل أو بوابات الدفع)، يُذكر صراحةً بوصفه **خارج النطاق / في قائمة الأعمال المؤجلة (Backlog)** دون مبالغة.

## الجمهور المستهدف

- المدقّقون الخارجيون والداخليون.
- الجهات الرقابية والتنظيمية.
- مسؤولو أمن المعلومات (CISO/Security Officers).
- فرق المطابقة وحماية البيانات.
