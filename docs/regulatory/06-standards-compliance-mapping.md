# 06 — مطابقة المعايير

## 1. الغرض والمنهجية

يربط هذا المستند **الضوابط الفعلية المطبَّقة في TFTSP** بمتطلبات معايير شائعة — بالأساس **ISO/IEC 27001:2022 (الملحق A)** ومبادئ **حماية البيانات الشخصية**. لكل ضابط: الوصف، أين طُبِّق (ملف/آلية)، والحالة. الحالات: **مطبَّق** / **مطبَّق جزئياً** / **إجراء تشغيلي** / **خارج النطاق (Backlog)**.

> **مبدأ الأمانة:** ما لم يُبنَ يُذكر صراحةً كـ Backlog، ولا يُقدَّم ضابط تشغيلي (مثل النسخ اليومي) على أنه ضابط مبرمَج.

## 3. مبادئ حماية البيانات الشخصية

| المبدأ | التطبيق في TFTSP | الحالة |
|---|---|---|
| تقييد الغرض | البيانات مقصورة على سياق شجرة الأنساب | مطبَّق |
| تقليل البيانات | عدم تخزين بيانات دفع؛ تأجيل حقول غير لازمة (D-308) | مطبَّق |
| سرّية البيانات | RLS + مُحلِّل الرؤية + التشفير | مطبَّق |
| التحكم بوصول صاحب البيانات | سياسات الرؤية (نساء/قُصَّر/متوفَّون) قابلة للضبط لكل قبيلة | مطبَّق |
| المساءلة | سجل تدقيق كامل بفرق before/after | مطبَّق |
| قابلية النقل (Portability) | تصدير `xlsx`/`csv` قابل للاستيراد | مطبَّق |
| الحق في المحو | حذف ناعم (`deleted_at`) قابل للاسترجاع | مطبَّق جزئياً (لا محو صلب مؤكَّد) |
| موافقة غير الأعضاء | طلبات المشاهدة بمرفق هوية + مراجعة إدارية | مطبَّق |

## 4. ملخّص العناصر خارج النطاق (Backlog) — بأمانة

العناصر التالية **غير مبنية** في v1 وتُذكر صراحةً حتى لا يُساء تفسير الحالة:

| العنصر | المرجع |
|---|---|
| المصادقة متعددة العوامل (MFA) | المواصفة §3 Backlog |
| نقل ملكية القبيلة الموثّق (تأكيد كلمة مرور + تدقيق `ownership_transfer`) | المواصفة §6 — غير مطبَّق في الكود |
| بوابات الدفع الإلكترونية (يوجد تفعيل يدوي/تحويل بنكي فقط عبر `ManualPaymentGateway`) | المواصفة §2، القرار D-402 |
| قنوات SMS/WhatsApp/Telegram (تجريد `NotificationChannel` جاهز للإضافة) | المواصفة §2، القرار D-205 |
| إدارة أسرار مركزية (Vault/KMS) | — |
| تخزين WORM / توقيع تشفيري لسجلات التدقيق / ختم زمني خارجي | راجع `04` §5 |
| فحص ثغرات آلي مستمر (SCA/DAST) وSIEM/تنبيه | راجع A.8.16 / A.12.6 |
| أرشفة WAL / نسخ عالي التردد / تجاوز فشل متعدد المناطق | راجع `05` §7 |
| المحو الصلب المؤكَّد للبيانات الشخصية | راجع §3 أعلاه |
| ميزات الذكاء الاصطناعي، GEDCOM، ODS، الشجرة الشعاعية/الزمنية، البيومتري، الروابط العميقة | المواصفة §3 Backlog |
